## One important addition

Ask the AI to **either provide or fetch the XCFramework**. For example, append this requirement:

> If `libtorrent.xcframework` is not already included in the project, either:
>
> * generate instructions for obtaining a compatible prebuilt XCFramework, or
> * configure the GitHub Actions workflow to download it from a trusted GitHub Release before building.
>
> Do not assume the framework already exists, and do not attempt to compile libtorrent from source during CI.

Without the actual `libtorrent.xcframework` binary (or a step to download it), the build will fail because Xcode will have nothing to link against.
