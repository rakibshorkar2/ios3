### One important note

Be cautious if an AI agent claims it can complete this end-to-end in a single pass. Integrating libtorrent on iOS involves building the C++ library for iOS, creating a Swift wrapper, and bridging it to Flutter. Those steps often require iterative debugging because of compiler, linker, and iOS SDK configuration issues. A more reliable approach is to have the agent implement and verify it in stages:

1. Build libtorrent for iOS.
2. Verify a native Swift app can download a known magnet link.
3. Add the Flutter bridge.
4. Integrate with your existing Downloads UI.

This staged approach makes it much easier to isolate and fix problems if they arise.
